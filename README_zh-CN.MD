# Go-Sci-hub-Back
### v0.1
Tampermonkey脚本：从sci-hub打开的链接返回到原始页面

Scihub的浏览器扩展程序将自动将Google Scholar页面重定向到Scihub页面。 但是，有时Scihub页面无法正常工作。

该脚本将在任何Scihub页面(\*.scihub.\*)的左上方添加一个按钮。 通过单击此按钮，您可以轻松地重定向回原始页面。
